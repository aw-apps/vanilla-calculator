# Agent Notes

## Project constraints
- Keep it **vanilla** (no frameworks, no build step, no runtime deps).
- App must work from repo root (GitHub Pages).
- Do **not** use `eval()` for expression evaluation.

## Quick validation checklist
- Open `index.html` and verify basic calculations: `1+2=3`, `2*3+4=10`, `10/4=2.5`.
- Keyboard: digits, operators, Enter, Backspace, Escape.
- Edge cases: repeated operators, decimal handling, divide-by-zero shows error and can be cleared.
